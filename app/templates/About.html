About the Dublin Bike Status Project

This project, created by Sangeetha and Pradyumna, MSc graduates in Business Analytics, harnesses data from an open API provided by data.gov.ie to monitor and report the status of Dublin Bikes. The primary goal is to collect real-time information, process it, and store it in a MySQL database. This approach ensures that data management is seamless, and all operational dependencies are adequately managed.

Project Objectives

  • To provide real-time status updates of Dublin Bikes across different stations.
  • To ensure accurate and timely data processing and storage.
  • To facilitate easy access to bike availability data for public use.

Technologies Used

The project is built using the following technologies:

  • Python for data fetching and processing.
  • MySQL for data storage.
  • PHP for server-side scripting.
  • HTML and CSS for frontend presentation.

Creators

Sangeetha and Pradyumna are passionate data enthusiasts with a keen interest in analytics and data science, particularly in the context of smart city initiatives. This project is a part of their commitment to leveraging technology for community benefits.
